== Description ==

TryAura is an AI-powered virtual try-on and content creation platform built for fashion and eCommerce. With seamless WooCommerce integration, it allows brands and shoppers to instantly visualize clothing on models, customers, or in any scene — without costly photoshoots. From product images to lifestyle videos, TryAura helps online stores boost engagement, reduce returns, and elevate their shopping experience with cutting-edge AI.

= Key Features =
* **AI Product Image Generation**: Generate high-quality product images from existing media using advanced AI models.
* **Video Generation**: Create short product showcase videos (Veo 3 via Gemini) to highlight your products.
* **Shopper-facing “Try on” Experience**: Injects a "Try on" button on WooCommerce product pages, allowing customers to visualize clothing using their own photos or camera.
* **Featured Image Enhancer**: An integrated tool in the WordPress Media Library to enhance featured images with AI-driven backgrounds and styles.

== Installation ==

1. Upload the `try-aura` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Navigate to the 'TryAura' menu in the admin dashboard to enter your Google AI API key.

== Frequently Asked Questions ==

= Where do I get an API key? =
You can get a Google AI API key from the Google AI Studio.

= Does it require WooCommerce? =
Yes, WooCommerce is required for the frontend "Try on" feature to appear on single product pages.

= Is it safe to use on production? =
Yes, but as with any AI-driven tool, it is recommended to review generated content before publishing.
